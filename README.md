# trab-1-atv-7
Esse código define uma estrutura chamada Livro, que armazena o título, o autor e o ano de publicação. Em seguida, o programa cria um vetor para armazenar 5 livros.
Primeiramente, ele realiza a leitura dos dados de cada livro, utilizando um laço. O título e o autor são lidos permitindo espaços, e o ano é lido como número inteiro.
Depois disso, o programa solicita ao usuário um título para busca. Em seguida, percorre todo o vetor de livros comparando o título digitado com os títulos armazenados, utilizando a função strcmp. Quando encontra uma correspondência exata, exibe os dados do livro e marca que houve resultado.
Por fim, caso nenhum livro seja encontrado, o programa informa isso ao usuário.
